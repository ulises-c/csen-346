# Project Selection Requirements

There are three options for the final project - topic is AI/NLP for Education/ Mentorship:

1. Select a paper from the ACL Anthology from the last 3 years (26-24) whose code is not available and implement the paper. Reproduce the results.
   1. https://aclanthology.org/
2. Select a paper from the ACL Anthology from the last 3 years whose code is available, and you are able to run it. Improve the solution by getting better results.
   1. https://aclanthology.org/
3. Propose your own novel project that involves NLP. Choose from the following ARR areas. Define what problem you are solving and how you intend to do it.
   1. https://aclrollingreview.org/areas
